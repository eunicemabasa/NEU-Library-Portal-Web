## 📖 Overview
The NEU Library Portal Web is a web-based system designed to help students and staff efficiently manage and access library resources. It provides an organized and user-friendly interface for browsing books, managing records, and improving overall library operations.

---

## ✨ Features
- 📚 Book catalog browsing  
- 🔍 Search and filter functionality  
- 👤 User-friendly interface  
- 🗂️ Organized library management system
